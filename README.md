# ⚡ BuildyBuddy — Build Without Code 🚀

**BuildyBuddy** is a no-code freelancing brand that helps businesses and individuals build powerful websites, tools, and automations — without writing a single line of code.

> Fast. Affordable. Stunning.

## 🌐 Live Preview

Open `index.html` in your browser to view the landing page.

## 📸 Features

### ✨ Interactive Elements
- 🖱️ **Cursor glow follower** — a subtle glow that follows your mouse (desktop)
- ⌨️ **Typing animation** — hero text cycles through 7 phrases
- 📊 **Animated counters** — numbers count up when scrolled into view
- 🃏 **3D tilt effect** — cards tilt toward the cursor with a shine overlay
- 📍 **Scroll progress bar** — gradient bar at the top of the page
- ⬆️ **Back to top button** — appears on scroll, smooth scroll to top
- 🎆 **Hero particles** — floating particles in the hero background
- 📱 **Mobile hamburger menu** — responsive nav for smaller screens
- 🎠 **Marquee animations** — two infinite-scrolling text/tool strips

## 🛠️ Tech Stack

| Technology | Usage |

## 🚀 Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/YOUR_USERNAME/BuildyBuddy.git
   ```

2. **Open in browser**
   ```bash
   cd BuildyBuddy
   open index.html
   ```
   Or simply double-click `index.html`.

3. **No build step needed** — it's pure HTML/CSS/JS!
